# Instagram Account Protocol

## Decide Account Type First
- Reels account
- Posts account

## Setup Steps
1. Set profile picture
2. Set bio
3. Create 2 posts on the main feed
4. Create a story highlight with a title
5. Set account to **Private**
6. Enable Authenticator (2FA)

## Output to Log for Each Instagram Account
- Username
- Email used
- Phone number used (if any)
- Account type: Reels or Posts
- Profile picture set: Yes / No
- Bio set: Yes / No
- Posts created: count
- Highlight title: text
- Private enabled: Yes / No
- 2FA enabled: Yes / No
- Backup codes saved: Yes / No  
  - Storage location

# Twitter Account Protocol

## Setup Steps
1. Create Twitter account on the phone
2. Set profile picture
3. Set bio
4. Set banner
5. Create 3 posts  
   - Each post: simple caption or no caption
6. Go to Messaging and create a PIN (example: 6677)
7. Enable Authenticator (2FA)
8. Save backup code **(mandatory)**

## Output to Log for Each Twitter Account
- Username
- Email used
- Phone number used (if any)
- Profile picture set: Yes / No
- Bio set: Yes / No
- Banner set: Yes / No
- Posts created: count
- Message PIN set: value + Yes / No
- 2FA enabled: Yes / No
- Backup codes saved: Yes / No  
  - Storage location

---

# Universal Rules (Instagram, Twitter, Snapchat)
- Always enable Authenticator (2FA)
- Always save backup codes, every time
- Log exactly where backup codes are stored  
  - Folder path or record ID
